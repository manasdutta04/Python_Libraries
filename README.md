# Python Libraries for AI/ML

This repository contains resources and examples for essential Python libraries used in Artificial Intelligence and Machine Learning.

## Libraries Covered

- **NumPy** - Fundamental package for numerical computing with Python
- **Pandas** - Data manipulation and analysis library
- **Matplotlib** - Comprehensive library for creating visualizations
- **Scikit-learn** - Machine learning library for classical ML algorithms
- **TensorFlow** - End-to-end platform for machine learning
- **PyTorch** - Deep learning framework
- **Keras** - High-level neural networks API
- **SciPy** - Library for scientific and technical computing

## Getting Started

Install the required libraries:

```bash
pip install numpy pandas matplotlib scikit-learn tensorflow torch keras scipy
